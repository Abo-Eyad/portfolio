# 🚀 AI-Generated Portfolio Template & Deployment Guide

Welcome! Building a professional, responsive, and deployment-ready portfolio shouldn't take weeks. This repository provides a complete, automated workflow to generate your own dark-themed, minimalist portfolio using any Large Language Model (LLM) and host it for free on GitHub Pages.

Whether you are an engineer, developer, or designer, you can use this "Super Prompt" to get your site live in minutes.

---

## 📂 Repository Structure

Here is a breakdown of the files included in this repository:

*   **`portfolio_super_prompt.md`**: The core AI prompt. It contains the instructions and the base HTML/CSS template. You will feed this to your AI of choice.
*   **`Portfolio_steps.md`**: The step-by-step guide (in both English and Arabic) on how to use the prompt and deploy the final code to GitHub Pages.
*   **`index.html`**: My personal portfolio file, generated using this exact method. You can use it as a live example.
*   **`profile.jpg` & `thumbnail.png`**: Example image assets required for the template. 

---

## 🛠️ How It Works

This project leverages Generative AI to handle the heavy lifting of web development. Instead of writing HTML and CSS from scratch, you simply provide your professional details to the AI, and it injects them into a pre-built, highly optimized template.

### Quick Start Guide

1.  **Prepare Your Assets:** Get a square profile picture (`profile.jpg`) and a landscape thumbnail (`thumbnail.png`) ready.
2.  **Open the Super Prompt:** Open `portfolio_super_prompt.md` and copy its entire contents.
3.  **Fill in Your Data:** Paste the prompt into an LLM (Claude, Gemini, ChatGPT, etc.) and replace the bracketed placeholders (e.g., `[Your Name]`, `[Your Bio]`) with your actual information.
4.  **Generate:** Hit send. The AI will output a complete, customized `index.html` file.
5.  **Deploy:** Follow the instructions in `Portfolio_steps.md` to push your new `index.html` file to GitHub and host it live for free!
