## **🌐 How to Generate and Deploy Your Portfolio (English)**

### **Phase 1: Using the Super Prompt**

1. **Gather Assets:** Prepare a professional headshot named profile.jpg (recommended 520x520 pixels) and a landscape preview image named thumbnail.png (recommended 1200x630 pixels).  
2. **Prepare the Prompt:** Copy the entire "Super Prompt" text provided in the template, along with the Source HTML at the bottom.  
3. **Fill the Blanks:** Replace all the bracketed placeholders under "User Variables" (like \[Your Name\], \[Your Email\], and project details) with your actual information.  
4. **Generate the Code:** Paste the filled prompt and the Source HTML into your preferred LLM (such as Gemini, ChatGPT, or Claude) and hit enter.  
5. **Save the File:** The AI will output a complete HTML code block. Copy this code and save it to your computer as a new file named exactly index.html.

### **Phase 2: Deploying to GitHub Pages**

1. **Create an Account:** If you do not already have one, sign up for a free account at GitHub.com.  
2. **Create a Repository:** Click the "+" icon in the top right corner and select "New repository".  
3. **Name the Repository:** Name the repository exactly yourusername.github.io (be sure to replace "yourusername" with your actual GitHub username). Leave it public.  
4. **Upload Files:** Open your new repository, click "Add file", and select "Upload files". Drag and drop your generated index.html file, along with your profile.jpg and thumbnail.png images.  
5. **Commit Changes:** Scroll down to the bottom of the page and click the green "Commit changes" button.  
6. **Go Live:** Wait a few minutes, then type \[https://yourusername.github.io/\](https://yourusername.github.io/) into your browser. Your new portfolio is now live on the internet\!

## **🌐 كيفية إنشاء ونشر محفظة أعمالك (عربي)**

### **المرحلة الأولى: استخدام الـ Super Prompt**

1. **تجهيز الـ Assets:** قم بإعداد صورة شخصية باسم profile.jpg (بأبعاد 520x520 بيكسل)، وصورة مصغرة للمحفظة باسم thumbnail.png (بأبعاد 1200x630 بيكسل).  
2. **نسخ الـ Prompt:** انسخ نص الـ Super Prompt بالكامل، بالإضافة إلى كود الـ Source HTML الموجود في الأسفل.  
3. **إدخال البيانات:** استبدل جميع الأقواس الموجودة تحت قسم "User Variables" (مثل \[Your Name\] و \[Your Email\]) ببياناتك وتفاصيل مشاريعك الحقيقية.  
4. **توليد الكود:** الصق الـ Prompt المكتمل مع الـ Source HTML في أي LLM تفضله (مثل Gemini أو ChatGPT أو Claude) واضغط إرسال.  
5. **حفظ الملف:** سيقوم الذكاء الاصطناعي بإنتاج كود HTML كامل. انسخ هذا الكود واحفظه على جهازك في ملف جديد باسم index.html.

### **المرحلة الثانية: النشر على GitHub Pages**

1. **إنشاء حساب:** إذا لم يكن لديك حساب، قم بالتسجيل مجاناً في موقع GitHub.com.  
2. **إنشاء Repository:** اضغط على علامة "+" في الزاوية العلوية واختر "New repository".  
3. **تسمية الـ Repository:** قم بتسمية الملف بالضبط yourusername.github.io (مع استبدال "yourusername" باسم المستخدم الخاص بك على GitHub). تأكد من أن الإعدادات على Public.  
4. **رفع الملفات (Upload):** افتح الـ Repository الجديد، اضغط على "Add file" ثم اختر "Upload files". قم بسحب وإسقاط ملف index.html الذي قمت بإنشائه، بالإضافة إلى صور profile.jpg و thumbnail.png.  
5. **تأكيد التعديلات (Commit):** انزل إلى أسفل الصفحة واضغط على الزر الأخضر "Commit changes".  
6. **تفعيل الموقع (Go Live):** انتظر بضع دقائق، ثم اكتب الرابط \[https://yourusername.github.io/\](https://yourusername.github.io/) في متصفحك. محفظة أعمالك أصبحت الآن متاحة على الإنترنت\!
